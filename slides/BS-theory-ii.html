Boij-Soederberg Theory

An $S$-module $M$ is called pure if there is a degree sequence ${\bf d} = (d_0 < d_1 < \cdots < d_n)$ such that $\beta_{i,j}(M)=0$ if $j \not = d_i$.

Notation: $\pi({\bf d})$ is the Betti diagram of a (the) pure module $M$ with the associated degree sequence ${\bf d} = (d_0 < d_1 < \cdots < d_n)$ with a (technical) scaling factor. Given two degree sequences ${\bf c}$ and ${\bf d}$, we say ${\bf c} \leq {\bf d}$ if $c_i \leq d_i$ for each $i$.
